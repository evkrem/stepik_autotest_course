Чтобы перейти в каталог репозитория, введите:
cd *имя репозитория *

Чтобы склонировать репозиторий к себе на компьютер введите в консоли:
git clone *адрес вашего репозитория*  

git help <команда> или git <команда> --help  если вы хотите посмотреть подсказку по какой-то конкретной команде
git - просмотр популярных команд
git status - позволяет отслеживать, какие изменения в репозитории вы сделали, что добавили в отслеживаемые файлы, что попало в коммит, что нет и так далее
git commit -m "тут ваше сообщение о коммите" - делаем коммит
git log - смотрим историю коммитов
git add name_file добавление файла
git push origin master - добавление изменений на сервер (пушим)
git pull - забираем с удаленного репозитория изменения на локальный комп

git branch newI - создание новой ветки с именем newI
git checkout newI - Эта команда перенесёт нас на новую ветку в момент, когда мы ещё не коммитили изменения
git checkout -b newI - создание новой ветки и переключение на неё с помощью одной команды
git merge newI - слияние ветки newI с мастером (при этом надо находиться в ветке мастер)
git rebase newI - слияние ветки newI с мастером (при этом надо находиться в ветке мастер)
git checkout HEAD^ - относительная ссылка - переход на родителя активного коммита
git checkout HEAD~3 - относительная ссылка - переход на 3 коммита вверх
git branch -f master HEAD~3  - переместит (принудительно) ветку master на три родителя назад от HEAD
git reset HEAD~1 - отменяет изменения, перенося ссылку на ветку назад, на более старый коммит (для локальных серверов).
git revert HEAD - делает новый коммит с изменениями, которыми можно поделиться с другими (для работы в команде)
git cherry-pick C2 C4 - копируем эти коммиты в место, где сейчас находишься (HEAD).
